# Taller de testing con React

Hola!

Bienvenid@ a este taller sobre cómo testear aplicaciones hechas con React. Abordaremos las pruebas unitarias 
con <Link text={'react-testing-library'} to={'https://www.npmjs.com/package/@testing-library/react'}/> y 
las pruebas *end to end* con <Link text={'Cypress'} to={'https://www.cypress.io/'}/>.

También nos basaremos en TDD (*Test Driven Development*) para diseñar y desarrollar nuevas funcionalidades a 
un proyecto ya existente.
